HTML Table Cheatsheet

Use these components to create a well laid out table

Element Description Example
<table> Defines a table <table>...</table>
<tr> Defines a table row <tr>...</tr>
<th> Defines a table header cell <th>Header</th>
<td> Defines a table data cell <td>Data</td>
<thead> Groups the header content in a table <thead>...</thead>
<tbody> Groups the body content in a table <tbody>...</tbody>
<tfoot> Groups the footer content in a table <tfoot>...</tfoot>
<caption> Defines a table caption <caption>Table Caption</caption>

The Final Result

Header 1 Header 2 Header 3

Add Math to a Table

Use the MathJax library to include mathematical notation in your table cells. For example:

Expression Description
\(E = mc^2\) Einstein's mass-energy equivalence formula
\(\int_a^b f(x) \,dx\) Definite integral of function \(f(x)\) from \(a\) to \(b\)
\(\frac{a}{b}\) Fraction representing division of \(a\) by \(b\)

Add Up Totals on a Table

Use JavaScript to calculate and display totals in your table. For example:

Item Quantity Price
Item A 2 10.00
Item B 3 15.00
Item C 1 7.50
Total $72.50
